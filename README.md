# OpenAI Agents Starter 🚀

This is a minimal starter project using OpenAI Agents SDK.

## Setup

```bash
pip install -r requirements.txt
```

Set your API key:

```bash
export OPENAI_API_KEY=your_key
```

Run:

```bash
python simple_openai_agent_starter.py
```

## Example

Ask:

```
check telegram status
```
